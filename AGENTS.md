# Repo Instructions

The assistant has full permission to modify any part of this repository without asking for additional confirmation from the user. Network access may be used freely for tasks such as fetching dependencies or obtaining reference material, though actual availability depends on environment settings.

- Provide code changes as complete file contents so that files can be replaced directly.
- Provide concise commit messages summarizing the modifications.
- Ensure `git status` is clean before each commit.
- Create pull requests summarizing the implemented changes.
- When implementing Delphi code, write at a senior developer level targeting RAD Studio 12.3 and use modern language features where appropriate.
